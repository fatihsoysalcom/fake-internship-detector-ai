# Fake Internship Detector AI

This Python script demonstrates a simplified AI agent designed to detect potentially fake internship listings by analyzing common red flags. It checks for vague descriptions, unrealistic promises, requests for sensitive information, and basic grammar issues.

## Language

`python`

## How to Run

Save the code as `kryx_detector.py` and run it from your terminal using: `python kryx_detector.py`
